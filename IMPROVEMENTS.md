## 🚀 Quick Start

### Backend (Terminal 1)
```bash
source venv/bin/activate
pip install -r server/requirements.txt
python server/app.py
```

### Frontend (Terminal 2)
```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:5173` in your browser!
